## 🇮🇹 Italiano

### Cos'è LINE?

LINE è un linguaggio di programmazione interpretato, progettato per essere comprensibile e facile da apprendere. Le keyword sono verbose e nominative — ogni costrutto dice esattamente cosa fa, rendendolo leggibile quasi come testo naturale.

Un programma LINE viene salvato in un file con estensione `.line` ed eseguito dall'interprete, disponibile nel browser senza installazioni.

## 🇬🇧 English

### What is LINE?

LINE is an interpreted programming language designed to be understandable and easy to learn. Its keywords are verbose and nominative — every construct says exactly what it does, making it readable almost like natural text.

A LINE program is saved in a file with the `.line` extension and executed by the interpreter, available in the browser with no installation required.

### Contribute

LINE is an open project. Contributions are welcome: bug reports, feature suggestions, code examples, or improvements to the manual.

→ [Open an issue on GitHub](https://github.com/qc-17/LINE/issues)
