Cos'è LINE

LINE è un linguaggio di programmazione progettato per essere comprensibile e facile da apprendere. La sintassi è pensata per essere leggibile e chiara anche per chi non ha mai programmato, mettendo comunque al centro le strutture logiche della programmazione. Dispone inoltre di un motore di rendering per equazioni (KaTeX), formule chimiche e integrazione con HTML e LINE-DOM.

Ogni file .line è eseguibile tramite interprete web o IDE integrato.

Caratteristiche

Sintassi italiana leggibile
Variabili, STAY, IF reattivi
Liste e cicli FOR
Funzioni FUN e blocchi THEN
Gestione errori TRY / SHOW
Moduli standard integrati
LINE-DOM per web

Obiettivo

LINE nasce per rendere la programmazione più accessibile, mantenendo però struttura e potenza di un linguaggio moderno.

È pensato per studenti, curiosi e sviluppatori che vogliono un approccio più intuitivo.
