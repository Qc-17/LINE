Cos'è LINE

LINE è un linguaggio di programmazione progettato per essere comprensibile e facile da apprendere. La sintassi è in italiano, verbosa e nominativa — ogni costrutto dice esattamente cosa fa.


Un file LINE si salva con estensione .line e viene eseguito dall'interprete, disponibile nel browser senza installazioni.

Caratteristiche

Sintassi in italiano, leggibile ad alta voce

Tre tipi di variabili: standard, costanti (STAY), reattive (IF)

Liste con indici, tag e iterazione con FOR

Funzioni con FUN, blocchi con THEN, condizioni con GO

Gestione errori con TRY / SHOW / YET

Moduli standard: math, str, list, rand, io e altri

Integrabile in HTML tramite LINE-DOM
